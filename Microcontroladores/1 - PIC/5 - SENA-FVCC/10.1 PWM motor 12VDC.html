Como ejemplo del uso de la señal PWM vamos a utilizar las funciones para controlar la velociidad de un motor de corriente continua conectado al colector de un transistor bipolar NPN (TIP41).

 

Haremos que nuestro circuito sea capaz de realizar un conteo ascendente/descendente entre 0 y 1000 y que éste valor sea asignado al ciclo de trabajo del PWM para manipular el voltaje aplicado a la base del transistor.

 

 

 

//#define P16F873A

//#define P16F883

//#define P16F877A

#define P16F887

//#define P18F2550

//#define P18F4550

 

#include ".\Drivers\Arduino.h.encrypted"

#include ".\Drivers\LiquidCrystal.h"

 

#define boton_inc PIN_A0   //Boton incremento

#define boton_dec PIN_A1   //Boton decremento

#define boton_reset PIN_A2 //Boton reset

 

int1 estado_boton_inc;

int1 estado_boton_dec;

int1 estado_boton_reset;

 

int16 contador = 0;

 

char txt2stream[12];

 

void setup()

{

   pinMode(PIN_A0, INPUT_);

   pinMode(PIN_A1, INPUT_);

   pinMode(PIN_A2, INPUT_);

   digitalWrite(LCD_BACKLIGHT, HIGH);

   lcd_begin(16,2);

   delay(200);

   sprintf(txt2stream,"PWM: 0");

   lcd_setCursor(0,0);

   lcd_print(txt2stream);

}

 

void loop()

{

   if (estado_boton_inc==1)

   {

      if (contador<1000)

      {

         contador += 10;

      }

   }

 

   if (estado_boton_dec==1)

   {

      if (contador>0)

      {

         contador -= 10;

      }

   }

 

   if (estado_boton_reset==1)

   {

      contador = 0;

   }

 

   sprintf(txt2stream,"%Lu ",contador);

   lcd_setCursor(5,0);

   lcd_print(txt2stream);

 

   analogWrite(PIN_C2,contador);

 

   delay(50);

}

 

Comentarios del programa:

 

La variable PWM se ha programado que su valor aumente o disminuya 10 unidades por cada pulsación. Se ha limitado el valor máximo del parámetro PWM a 1000 y el mínimo a 0 y que nunca rebase estos límites. .

 

Se debe tener especial atención al asignar el ciclo de trabajo al PWM, el cual debe ser un número entero comprendido entre 0 (0V) y 1000 (5V), es decir, el incremento en una unidad en el contador se verá reflejado como una variación de 0,05V a la salida del pin C2 del microcontrolador.
