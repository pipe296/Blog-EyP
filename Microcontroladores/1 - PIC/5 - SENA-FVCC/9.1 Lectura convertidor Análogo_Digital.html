Lectura convertidor Análogo/Digital

 

Como ejemplo del uso del conversor ADC del microcontrolador vamos a utilizar las funciones para leer el dato proveniente de un sensor en el rango de 0 a 5V y representar en un display LCD el valor leído por el convertidor ADC en el rango de 0 a 1023.

 

 

//#device PIC16F84A

//#device PIC16F628A

//#device PIC16F818

//#device PIC16F873A

//#device PIC16F883

//#device PIC16F877A

#device PIC16F887

//#device PIC18F2550

//#device PIC18F4455

//#device PIC18F4550

 

#include <Arduino.h.encrypted>

#include <LiquidCrystal.h>

 

long sensor;

 

char txt2stream[10];

 

void setup() {

   digitalWrite(LCD_BACKLIGHT, HIGH);

   lcd_begin(16,2);

   delay(200);

   lcd_setCursor(0,0);

   lcd_print("Sensor:");

}

 

void loop() {

   sensor = analogRead(A0);

   sprintf(txt2stream,"%Lu   ",sensor);

   lcd_setCursor(8,0);

   lcd_print(txt2stream);

   delay(500);

}

 

Comentarios del programa:

 

El ejemplo está debidamente documentado, por lo cual su comprensión no reviste mayor dificultad.

 

Se debe tener especial atención al seleccionar el canal a leer:

 

analogRead(A0); esta instrucción selecciona el canal AN0

analogRead(A1); esta instrucción selecciona el canal AN1

analogRead(A2); esta instrucción selecciona el canal AN2…  etc…

 

Esta instrucción lee el valor análogo en el canal seleccionado previamente

 

Para observar la variación de la unidad medida se ha dispuesto un potenciómetro como sensor.
